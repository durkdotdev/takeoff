# Tailwind CSS

[Tailwind CSS](https://tailwindcss.com/) is a highly popular, utility-first CSS framework.

If you selected Tailwind CSS, TAKEOFF will automatically configure it within `_app.tsx`:

```jsx
// _app.tsx

import "tailwindcss/tailwind.css";

...
```
